Project Name: Techspecs

Description:
This project is a responsive landing page converted from the provided design using clean HTML and CSS.
The layout is fully responsive and works well on desktop, tablet, and mobile devices.

Folder Structure:
- index.html        → Main HTML file
- css/master.css     → Main stylesheet
- images/           → Project images

How to Use:
1. Extract the ZIP file.
2. Open index.html in any modern web browser.
3. No additional setup or dependencies are required.

Customization:
- You can edit text content directly in index.html.
- Colors, fonts, and layout styles can be adjusted in css/master.css.
- Images can be replaced inside the images folder (keep the same file names).

Responsiveness:
The website is fully responsive and adapts to different screen sizes.
